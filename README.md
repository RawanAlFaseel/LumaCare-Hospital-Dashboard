# Project Overview

A hospital finance dashboard designed to provide a clear overview of billing and cost performance.
The dashboard highlights key financial insights such as medication costs, room charges, and billing trends across departments and procedures.

- Dataset <br>
The dataset is open and available from the (Data with Decision) YouTube channel.  [https://acesse.one/TC59K]
